Button Preview Matrix

This view shows every supported button family across appearances, sizes, and interaction states at once. Text and icon-only variants are both included where the component supports them.

Families: standard, menu, split
Sizes: large, small, compact
States: default, hover, focus, active, disabled
Split focus: action half and chevron half shown separately
Appearances: primary, secondary, outline, subtle, transparent

Hover, focus, and active are visual simulations in this static page so you can compare combinations side by side without interacting with each example.
